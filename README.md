# MyGameSpaceInvaders
Space Invaders на C# + WinForms
#### Управление:
- Движение игрока враво - key_Right 
- Движение игрока влево- key_Left		
- Стрельба - key_Space
- Поставить на паузу / Продолжить - key_Escape
- Начать игру - key_S
- Перезапустить игру после проигрыша - key_R
- Следующий уровень - key_N
